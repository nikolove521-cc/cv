# 欧捷科技 · 抖音获客项目复盘报告

## 一、项目背景

- **项目周期**：2025年4月 - 2026年6月
- **项目阶段**：新业务市场验证期 → 规模化投放期
- **投放平台**：抖音（信息流+搜索双引擎）
- **核心目标**：通过抖音渠道获取有效销售线索，验证AI营销服务的市场需求与获客模型

---

## 二、投放策略制定

### 1. 策略框架

```
目标人群定位 → 素材规划 → 投放测试 → 数据监测 → 阶段复盘 → 策略迭代
```

### 2. 核心策略

| 策略维度 | 具体动作 |
|---|---|
| **定向策略** | 基于行业标签（制造/商贸/服务类企业）+ 行为兴趣（B2B营销/获客/CRM）+ 关键词搜索三重定向 |
| **素材策略** | A/B测试3类素材方向：痛点场景型 / 数据案例型 / 产品展示型，每周更新素材库 |
| **出价策略** | 初期采用oCPM智能出价快速获取数据，中期转为CPC手动出价控制成本，后期ROI回正后逐步放量 |
| **落地页策略** | 针对不同行业定制落地页模板，强化"AI获客"核心卖点，设置表单+电话双转化入口 |

### 3. 资源协调

- 与销售团队建立SOP流程，明确线索分配时效（2小时内跟进）
- 与内容团队协作，建立素材生产标准化流程
- 建立数据跟踪机制，每日监控线索质量及转化漏斗

## 三、数据表现（核心指标）

### 投放效果总览

| 指标 | 优化前 | 优化后 | 变化幅度 |
|---|---|---|---|
| **单条线索成本** | 基准值（设为100%） | ↓ 68% | **-68%** |
| **整体投产比（ROI）** | — | **1.9** | 稳定达标 |
| **月均线索量** | 基准值 | ↑ 22.5% | **+22.5%** |
| **线索有效率（峰值）** | — | **88.9%** | 阶段性高点 |
| **月均线索有效率** | — | **≥60%** | 持续稳定 |

### 漏斗分析

```
曝光 → 点击 → 落地页停留 → 表单提交 → 有效线索 → 销售跟进 → 成交转化
 100%   3.2%       45%         8%          60%           35%          12%
```

> 注：线索有效率 = 有效线索数 / 总线索数；销售跟进转化率 = 成交数 / 有效线索数

## 四、关键优化动作

### 优化阶段一：定向精准化（第1-2个月）

- **问题**：初期定向过宽，无效线索占比高
- **动作**：
  - 收窄行业定向，聚焦高转化行业（制造业、商贸流通、企业服务）
  - 排除低意向人群包（学生、自由职业者等）
  - 增加搜索关键词否定词库
- **效果**：线索有效率从42%提升至65%

### 优化阶段二：素材效率提升（第3-4个月）

- **问题**：素材疲劳导致CTR下降，成本上升
- **动作**：
  - 建立每周素材更新机制，保持3-5组新素材并行测试
  - 优化落地页加载速度，减少跳出率
  - 针对高CTR素材类型加大投放比例
- **效果**：CTR从1.8%提升至3.2%，单条线索成本下降35%

### 优化阶段三：ROI精细化运营（第5-6个月）

- **问题**：线索量增长但ROI波动较大
- **动作**：
  - 建立日度数据看板，实时监控各计划ROI表现
  - 对ROI<1.5的计划及时关停或调整出价
  - 与销售团队对齐线索质量标准，反向优化投放定向
- **效果**：ROI稳定在1.9，线索成本累计下降68%

## 五、项目成果总结

### 量化成果

1. ✅ 线索成本降低 **68%**，大幅优于行业平均水平
2. ✅ 投产比稳定在 **1.9**，实现可持续获客模型
3. ✅ 月均线索量增长 **22.5%**，在成本下降的同时实现量增
4. ✅ 建立完整的投放-SOP-转化闭环流程

### 沉淀资产

- 一套可复用的抖音投放策略框架
- 一个经过验证的素材库（3类方向 × 多版本）
- 一套数据监控与复盘SOP
- 与销售团队的线索流转协同机制

## 六、经验与反思

### 成功经验

- **数据驱动决策**：每日数据监控+周度复盘，快速识别问题并调整
- **闭环思维**：不仅关注投放端，更关注线索后续转化，形成双向优化
- **SOP建设**：建立标准化的投放-跟进-复盘流程，降低对个人经验的依赖

### 待优化方向

- 自然流账号引流不足，过度依赖付费投放
- 未建立长效的品牌内容矩阵，获客模式偏短期
- AI产品本身还在孵化期，部分线索因产品成熟度不足未能转化
