# 公司评估 Checklist

> 面试前或收到 offer 时，用这个 checklist 快速判断公司靠不靠谱。
> 每个维度 1-5 分，总分 ≥ 18 分可以放心接，12-17 分需要权衡，< 12 分谨慎。

---

## 一、基本信息调查（用天眼查/企查查看）

| 评估项 | 评分标准 | 得分(1-5) |
|---|---|---|
| 成立时间 | ≥3年得5分，1-3年得3分，<1年得1分 | |
| 注册资本 | ≥100万得5分，50-100万得3分，<50万得1分 | |
| 参保人数 | ≥50人得5分，10-50人得3分，<10人得1分 | |
| 融资情况 | 有多轮融资得5分，天使轮得3分，无融资得1分 | |
| 司法风险 | 无诉讼得5分，少量劳动纠纷得3分，多起诉讼得1分 | |
| 经营异常 | 无异常得5分，有异常已修复得3分，未修复得1分 | |

## 二、面试感受评估

| 评估项 | 评分标准 | 得分(1-5) |
|---|---|---|
| 面试官专业度 | 聊业务细节得5分，只谈画饼得2分 | |
| 团队氛围 | 团队活跃、有笑声得5分，冷场尴尬得2分 | |
| 办公环境 | 正规写字楼得5分，共享办公得3分，居民楼得1分 | |
| 岗位描述清晰度 | JD和面试一致得5分，完全不一样得1分 | |
| 入职流程规范性 | 当场发offer得5分，拖泥带水得2分 | |

## 三、薪资福利评估

| 评估项 | 评分标准 | 得分(1-5) |
|---|---|---|
| 薪资结构 | 底薪明确+绩效透明得5分，全部绩效得1分 | |
| 五险一金 | 全额缴纳得5分，最低基数得3分，不缴得0分 | |
| 加班文化 | 不加班/偶尔得5分，单休得2分，大小周得3分 | |
| 发展空间 | 有明确晋升路径得5分，说不清楚得2分 | |

## 四、Boss 直聘上的隐藏信息

- **HR 活跃度**：今天活跃 > 3天活跃 > 一周前活跃（越久越不靠谱）
- **公司正在招聘的岗位数量**：一次性招10+个同类岗位 = 高流失率预警
- **Boss 直聘企业认证**：已认证的 > 未认证的
- **公司规模标签**：100-499人 > 20-99人 > 少于20人

## 五、红旗信号 ⚠️

出现以下任一情况，建议谨慎：
- [ ] 面试不问专业技能，只问"什么时候能到岗"
- [ ] 薪资结构说不清楚，或"底薪+绩效"比例不合理（绩效占比超过40%）
- [ ] 试用期不发社保/不全额缴纳
- [ ] 要求你先交培训费/服装费/押金（100%是骗子）
- [ ] 办公地点在居民楼/地下室
- [ ] 公司名称和签合同的公司不一致
- [ ] 说"能力强可以拿更高薪资"但底薪很低

## 六、绿灯信号 ✅

出现以下情况，基本靠谱：
- [ ] 面试流程规范，有笔试/实操环节
- [ ] 愿意详细介绍团队架构和工作内容
- [ ] 主动说清楚薪资结构和考核标准
- [ ] 办公环境正规，团队看起来有活力
- [ ] 能在 Boss 直聘上看到其他员工的评价
