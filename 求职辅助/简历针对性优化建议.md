## 二、不同岗位方向的简历调整

### 方向一：广告投放 / 增长运营（推荐首选）

**调整重点**：
- 把"线索成本降低68%"、"投产比1.9"放在每个工作经历的业绩最前面
- 欧捷科技那段，强调"从0到1搭建投放体系"和AI营销项目经验
- 删除或弱化与客户对接相关的内容
- 突出 Excel 数据处理、数据建模能力

**个人优势改为**：
```
- 9年互联网广告投放与增长经验，精通腾讯/字节/百度/抖音等平台
- 数据驱动增长：线索成本↓68%，投产比稳定1.9，线索量↑22.5%
- 全链路投放：从策略制定→素材规划→数据监测→ROI优化闭环
```

### 方向二：广告执行 / 客户运营

**调整重点**：
- 把辉煌明天那段放在最前面（最相关）
- 强调"30+客户"、"2000万流水"、"100%按时上线"
- 突出跨部门协调能力
- 保留客户满意度、续费率的成果

**个人优势改为**：
```
- 9年广告行业经验，擅长客户管理与项目统筹
- 服务30+活跃客户，半年累计广告流水2000万+，项目100%按时交付
- 客户运营：满意度提升30%，续费率增加25%，季度消耗流水2000万+
```

### 方向三：渠道运营

**调整重点**：
- 把潮生活那段（公众号渠道管理）往前放
- 突出"200+渠道管理"、"75%盈利占比"、"人力成本降低30%"
- 强调 SOP 流程建设和数据分析能力

**个人优势改为**：
```
- 9年互联网渠道与新媒体运营经验，擅长数据驱动增长
- 渠道管理：管理200+引流渠道，优化后75%以上账号盈利
- 运营提效：建立SOP流程，人力成本降低30%，客户满意度提升30%
```
