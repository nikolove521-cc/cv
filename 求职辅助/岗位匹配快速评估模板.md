# 岗位匹配快速评估模板

> 使用方法：在 Boss 直聘上看到感兴趣的岗位，把 JD 粘贴进来，按下面的维度快速打分。
> 总分 ≥ 30 分 → 优先投，话术选版本 A/B/C
> 总分 20-29 分 → 可以投，用版本 D 通用版
> 总分 < 20 分 → 不建议投，匹配度太低浪费机会

---

## 匹配评分表

| 评估维度 | 权重 | 得分 (0-10) | 说明 |
|---|---|---|---|
| **1. 行业经验匹配** | 20% | | 是否在互联网广告/营销/运营领域？ |
| **2. 核心技能匹配** | 25% | | 是否要求投放优化、数据分析、渠道管理？ |
| **3. 平台经验匹配** | 15% | | 是否要求腾讯/字节/百度/抖音等平台经验？ |
| **4. 业绩导向** | 20% | | 是否强调ROI、线索量、转化率等数据指标？ |
| **5. 薪资范围匹配** | 10% | | 是否在你的 12-15K 范围内？ |
| **6. 公司规模/稳定性** | 10% | | 公司规模、成立时间、融资情况 |
| **总分** | 100% | | |

---

## 加分项（额外 +5 分/项）

- [ ] 公司有 AI 营销/增长黑客/数字化转型相关业务
- [ ] 岗位有明确的晋升通道或团队管理职责
- [ ] 公司是行业头部或知名初创
- [ ] 团队规模 20 人以上（说明岗位不是临时顶替）

## 减分项（额外 -5 分/项）

- [ ] JD 写得很模糊，说不清具体做什么
- [ ] 要求"全能型"人才（既要设计又要投放又要客服）
- [ ] 公司规模小于 10 人且无融资信息
- [ ] 薪资范围低于 12K 或高于 15K 太多

---

## 快速判断口诀

**一看行业**：互联网广告/营销/电商优先，传统行业广告部慎投
**二看技能**：投放+数据+客户对接，三者占其二就值得投
**三看数据**：JD 里写具体指标的岗位，说明他们重视结果，你也容易出成绩
**四看公司**：企查查/天眼查搜一下，成立时间<1年的谨慎，有法律诉讼的要留意
**五看薪资**：12-15K 区间，如果对方给 12K 以下，除非平台好可以考虑

---

## 岗位优先级分类

### 🔴 S级（优先投，当天就投）
满足以下条件中的 3 个及以上：
- 明确要求 3-5 年以上广告/运营经验
- 提到腾讯/字节/百度/抖音等平台
- 强调数据驱动、ROI、转化率
- 薪资 13K 以上

### 🟡 A级（重点投，本周内投）
满足以下条件中的 2 个及以上：
- 要求 2 年以上相关经验
- 互联网/广告/营销相关行业
- 薪资 12-15K

### 🟢 B级（有空投，备选）
- 经验要求不高但薪资偏低
- 行业不太匹配但有学习空间
- 公司规模小但不确定好坏

### ⚪ C级（不投）
- 完全不符合你的经验方向
- 薪资远低于预期
- 明显是销售岗包装成运营岗
