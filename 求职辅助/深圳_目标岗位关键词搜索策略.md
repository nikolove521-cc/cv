```

效果广告 + 深圳
增长运营 + 深圳
广告执行 + 深圳
渠道运营 + 深圳
新媒体运营 + 深圳
数字营销 + 深圳
```

### 2. 筛选条件设置
- **薪资范围**：10-15K（留一点谈判空间）
- **经验要求**：3-5年 / 5-10年（避开1-3年，你的经验更丰富）
- **学历要求**：不限 / 大专及以上（避开本科起步的岗位）
- **最新发布**：勾选"3天内发布"

### 3. 隐藏关键词（JD 里可能有但搜索名不同的）
```
用户增长、私域运营、信息流投放、信息流优化师
SEM优化师、SEO运营、内容运营、媒介策划
```

## 深圳热门行业 & 适合你的公司类型

### 互联网/广告科技公司
- 南山区：科技园片区（腾讯周边生态企业最多）
- 福田区：CBD片区（广告公司、传媒公司集中）
- 宝安区：跨境电商公司多（需要投放运营）

### 推荐关注的企业类型
1. **广告代理商**：蓝色光标、省广集团、华扬联众等
2. **互联网大厂生态**：腾讯/字节/阿里服务商
3. **跨境电商公司**：深圳特色，大量需要广告投放运营
4. **SaaS/营销工具公司**：如微盟、有赞等
5. **MCN/新媒体公司**：内容运营+投放需求大
