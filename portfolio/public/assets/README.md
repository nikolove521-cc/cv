# 替换占位内容的说明

## 1. 个人照片
将你的个人照片放入此目录，命名为 profile.jpg，然后修改 About.jsx。

## 2. 项目图片
将项目截图/PPT案例放入此目录，命名为 project-1.jpg 等。

## 3. Hero 视频背景
将视频文件放入此目录，命名为 hero-bg.mp4，然后修改 Hero.jsx。

## 推荐格式
- 照片: JPG/WebP, 约 800x1067px (3:4)
- 项目图: JPG/WebP, 约 1200x750px (16:10)
- 视频: MP4 (H.264), 约 10-15MB, 1920x1080
